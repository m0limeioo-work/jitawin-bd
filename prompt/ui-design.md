# JitaWin 代理推廣網站 UI Design 推薦

**版本**：1.0  
**目標**：高轉換率、行動優先、孟加拉市場信任感 + 興奮氛圍  
**最後更新**：2026-07-13

## 1. 整體設計原則
- **色系主調**：
  - 主色：深綠 (#006400 或 #008000) — 代表幸運與板球草地
  - 輔助色：金黃 (#FFD700 / #FFCC00) — 獎金、勝利
  - 背景：深黑 / 深灰 (#0F172A 或 #111827)
  - 強調色：亮綠或橙色（CTA 按鈕）
- **字體**：英文 - Inter / Roboto；孟加拉文 - Noto Sans Bengali
- **風格**：現代暗黑奢華、遊戲化（gamification）、高對比
- **佈局**：Mobile-First 單頁滾動（One Page Landing），桌面版適配
- **氛圍**：興奮、專業、可信任（避免過度浮誇）

## 2. 核心畫面結構

### Hero Section（首屏）
- 全屏背景：板球比賽動態圖 + 老虎機光效疊加
- 大標題（孟加拉文粗體）：  
  「JitaWin Bangladesh - ক্রিকেট বেট করে জিতুন লাখ টাকা！」
- 子標題：快速 Bkash/Nagad 存款、最低 ৳100、首存大禮
- CTA：兩個大按鈕（金色 + 綠色）
  - 「এখনই রেজিস্টার করুন」→ 代理註冊連結
  - 「APP ডাউনলোড করুন」
- 下方信任徽章：牌照、支付 Logo、用戶數

### Navigation Bar（固定導航）
- 左側 Logo（JW 深綠底黃字）
- 選單（孟加拉文）：স্লট, লাইভ, স্পোর্ট, ফিশার, প্রমোশন, VIP, খবর, রেফারেল
- 右側：登入 / 註冊按鈕 + 語言切換

### 主要內容區塊
1. **為什麼選擇 JitaWin**（Trust Bar）
   - 卡片網格：合法牌照、快速提款、本地支付、安全保障

2. **熱門遊戲分類**
   - 橫向或網格卡片，含遊戲預覽圖 + 供應商標誌
   - 重點強調板球投注與熱門老虎機

3. **最新優惠 / 促銷**
   - 大卡片 + 倒數計時（如果有時限活動）
   - 表格呈現儲值加碼、返水

4. **VIP 會員制度**
   - 進度條視覺化 + 等級表格（高亮重點等級）

5. **推薦好友賺佣金**
   - 專屬 QR Code + 推薦碼展示區
   - 佣金比例圖示

6. **如何開始（Step by Step）**
   - 3-4 步驟卡片：註冊 → 驗證 → 存款 → 領獎

7. **App 下載專區**
   - PWA 安裝教學 + 手機模擬畫面

8. **用戶見證 / 成功故事**
   - 輪播卡片（真實感孟加拉使用者故事）

9. **FAQ**（手風琴式）

### Footer
- 簡潔連結 + 責任博弈提醒 + 聯絡資訊

## 3. UI 元件推薦
- **按鈕**：圓角、大尺寸、金色/綠色漸層，hover 發光動畫
- **卡片**：深色背景、微陰影、懸停放大 + 邊框亮光
- **表格**：清晰斑馬紋、行動端可橫向滾動
- **圖示**：線性圖標 + 支付 Logo + 遊戲類別 Icon
- **動畫**：
  - CTA 脈衝
  - 數字計數（獎金金額）
  - 滑入動畫（Scroll Reveal）

## 4. 行動端特別優化
- 底部固定 CTA Bar
- 大觸控目標
- 簡化選單（漢堡）
- 垂直內容堆疊
- 快速加載（圖片優化）

## 5. 參考靈感風格
- 暗黑現代博弈網站（如 Cloudbet、1xBet 風格）
- 板球主題本地化元素（綠地、國家色）
- 高轉換 Landing Page 最佳實務
